# Portfolio Website

Personal portfolio showcasing my work as a Computer Science student at Western University.

## Features

- **Modern Design** - Dark theme, custom cursor, and floating particles
- **Interactive Rubik's Cube** - Fully functional 3D Rubik's Cube with rotation controls and scramble functionality
- **Responsive Layout** - Works on desktop and mobile devices
- **Smooth Animations** - Typing effects, scroll reveals, and tilt effects on project cards

## Tech Stack

- HTML5
- CSS3 (Custom properties, animations, 3D transforms)
- Vanilla JavaScript

## Sections

- **Hero** - Introduction with animated typing effect
- **About** - Skills and background info
- **Projects** - Featured work with links to GitHub repos
- **Rubik's Cube** - Interactive 3D puzzle
- **Contact** - Contact form and social links

## Running Locally

Just open `index.html` in a browser, or use a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`
